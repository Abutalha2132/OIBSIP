About This Repository

This repository contains all the projects completed during my internship
with Oasis Infobyte under the OIBSIP program.

The projects focus on data analytics, data cleaning, visualization,
and machine learning using Python and Jupyter Notebook.



Projects Included

1. Exploratory Data Analysis (EDA) on Retail Sales Data  
    Dataset: retail_sales_dataset.csv  
    Analyzed sales trends, customer behavior, and product performance.

2. Customer Segmentation Analysis  
    Dataset: menu.csv  
    Used clustering techniques to segment customers based on behavior.

3. Predicting House Prices with Linear Regression  
    Dataset: Housing.csv  
    Built a linear regression model to predict house prices.

4. Sentiment Analysis  
    Dataset: Twitter_Data.csv  
    Performed sentiment analysis on Twitter data using NLP techniques.

5. Fraud Detection  
    Dataset: creditcard.csv  
    Detected fraudulent transactions using machine learning models.

6. Data Cleaning  
    Dataset: AB_NYC_2019.csv  
    Cleaned and prepared real-world data for analysis.

7. Wine Quality Prediction  
    Dataset: WineQT.csv  
    Predicted wine quality using classification models.
